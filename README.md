# 게시판 서비스

## 개요
에브리타임 디자인을 클론코딩하여 만들었습니다.

**일정**
- 24.08 ~
- 개인 프로젝트

## 사용 기술 및 개발 환경
- O/S : macOS 13.4.1 (Ventura)
- Server : Apache Tomcat 8.5
- DB : H2
- Framework : Spring framework, JPA
- Language : JAVA
- Tool : IntelliJ, GitHub, Git(SourceTree)

## 구현 기능
### 게시판
- 게시글 및 댓글 생성
- 게시글 및 댓글 조회

### 회원 관리
- 회원가입
  - 유효성 검사
